## API

| 属性 | 类型 | 默认 | 必要 | 描述 |    
| ---| ---  | --- | --- | ----|
| placeholder | string |  搜索     | true | 占位文字 |
| options     | any[]  | [] | - | 数据 |
| onSelect    | (value: string, option: any) => void | - | - | 选择回调 |
